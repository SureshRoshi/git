# Create a new repository on the command line

```bash
echo "# git" >> README.md
git init
git add .
git commit -m "first commit"
git branch -M main
git remote add origin -- github repo link --
git push -u origin main
```

# Push an existing repository from the command line

```bash
git remote add origin -- github repo link --
git branch -M main
git push -u origin main
```
